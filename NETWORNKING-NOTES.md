# Networking Basics

## What is an IP Address?
An IP address identifies a device on a network.

## Types of IP Addresses
- Public IP
- Private IP

## Common Ports
- 80 → HTTP
- 443 → HTTPS
- 22 → SSH

## OSI Model(seven layers)
1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application
